<pre>+---+

+---+

| 8 |
| 3 |
| 3 |
| 1 |
| 4 |
| 5 |
| 6 | 
</pre>

<p>对于上面给出的样例数据，你的查询语句应该返回如下结果：</p>

<pre>+---+
|num|
+---+
| 6 |
</pre>
