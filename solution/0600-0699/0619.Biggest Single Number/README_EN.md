## Description

Can you write a SQL query to find the biggest number, which only appears once.</p>

+---+

|num|

+---+

| 8 |

| 8 |

| 3 |

| 3 |

| 5 |

</pre>

For the sample data above, your query should return the following result:

<pre>

+---+

|num|

+---+

| 6 |

</pre>

<b>Note:</b><br />

If there is no such number, just output <b>null</b>.
