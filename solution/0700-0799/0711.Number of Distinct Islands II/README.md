<p>给定一个非空01二维数组表示的网格，一个岛屿由四连通（上、下、左、右四个方向）的 <code>1</code> 组成，你可以认为网格的四周被海水包围。</p>

<p>请你计算这个网格中共有多少个形状不同的岛屿。如果两个岛屿的形状相同，或者通过旋转（顺时针旋转 90&deg;，180&deg;，270&deg;）、翻转（左右翻转、上下翻转）后形状相同，那么就认为这两个岛屿是相同的。</p>
