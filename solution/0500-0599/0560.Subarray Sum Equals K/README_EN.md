## Description
<p>Given an array of integers and an integer <b>k</b>, you need to find the total number of continuous subarrays whose sum equals to <b>k</b>.</p>



<p><b>Example 1:</b><br />

<pre>

<b>Output:</b> 2

</pre>

</p>

</ol>

</p>
