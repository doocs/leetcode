<p>给你一个已经<strong>&nbsp;排好序</strong>&nbsp;的整数数组&nbsp;nums&nbsp;和整数&nbsp;a、b、c。对于数组中的每一个数 x，计算函数值&nbsp;f(<em>x</em>) = <em>ax</em><sup>2</sup> + <em>bx</em> + c，请将函数值产生的数组返回。</p>
