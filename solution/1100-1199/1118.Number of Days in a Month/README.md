<p>指定年份 <code>Y</code> 和月份 <code>M</code>，请你帮忙计算出该月一共有多少天。</p>

<pre><strong>输入：</strong>Y = 2000, M = 2
<strong>输出：</strong>29

<pre><strong>输入：</strong>Y = 1900, M = 2
<strong>输出：</strong>28
</pre>

	<li><code>1 <= M <= 12</code></li>
