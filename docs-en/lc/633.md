# [633. Sum of Square Numbers](https://leetcode.com/problems/sum-of-square-numbers)



## Description

<p>Given a non-negative integer <code>c</code>, decide whether there&#39;re two integers <code>a</code> and <code>b</code> such that <code>a<sup>2</sup> + b<sup>2</sup> = c</code>.</p>

<p>&nbsp;</p>
<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> c = 5
<strong>Output:</strong> true
<strong>Explanation:</strong> 1 * 1 + 2 * 2 = 5
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> c = 3
<strong>Output:</strong> false
</pre>

<p>&nbsp;</p>
<p><strong>Constraints:</strong></p>

<ul>
	<li><code>0 &lt;= c &lt;= 2<sup>31</sup> - 1</code></li>
</ul>

## Solutions

### Solution 1

=== "Python3"

    ```python
    class Solution:
        def judgeSquareSum(self, c: int) -> bool:
            a, b = 0, int(sqrt(c))
            while a <= b:
                s = a**2 + b**2
                if s == c:
                    return True
                if s < c:
                    a += 1
                else:
                    b -= 1
            return False
    ```

=== "Rust"

    ```rust
    use std::cmp::Ordering;
    impl Solution {
        pub fn judge_square_sum(c: i32) -> bool {
            let c = c as i64;
            let mut left = 0;
            let mut right = (c as f64).sqrt() as i64;
            while left <= right {
                let num = left * left + right * right;
                match num.cmp(&c) {
                    Ordering::Less => {
                        left += 1;
                    }
                    Ordering::Greater => {
                        right -= 1;
                    }
                    Ordering::Equal => {
                        return true;
                    }
                }
            }
            false
        }
    }
    ```
